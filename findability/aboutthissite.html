About This Site

1. Semantic HTML

• h1, h2 tags - A header tags are much like the subject line of your web page. I use collect keyword for h1 tag on each page.

• Validate HTML, CSS - It is important to improve usability and functionality. I made website run into no errors and user-friendly.

• W3C industry standards - Use HTML elements for their designated purposes that make search engines to find your site easier and give high lancing.

• Footer nav for mobile – It allows users to quickly move to a different section and back to page what they want. It is also important to create user-friendly mobile version site to be picked by search engine.

• Responsible design - The responsiveness of your website is unequivocally an important factor in improving user experience and avoiding common SEO pitfalls that can hinder your Google rankings.

2. Accessibility (WCAG)

• Table - to increase accessibility of my data table, I made it read-friendly for screen reader.

• Hcard - Hcard is a microformat for publishing the contact details of people, companies, organizations, and places, in HTML, Atom, RSS, or arbitrary XML. I used it for my form.

• Skip nav - It is also good to increase your accessibility to help people who will use screen reader.

3. Images optimisation

• Image size – I optimise all of images on my website with right compression rate.

• File format – I picked the right file format for every images.

• Optimise image file names – I gave file names which explain the images exactly.

• Use alt tags for optimise image title – it is important for findability and for accessibility. I use alt tags for all of images on my website.

4. Secure staffs

• htaccess – It provide a way to make configuration changes on a per-directory basis.

• robots.txt – It is used mainly to avoid overlording my site with requests.

5. Sitemap Google search console

• Sitemap – It is a file where I provide information about the page, videos, and other files on my site, and the relationship between them. It is important because search engines read this file to crawl my site more efficiently.

6. Content

• Keyword – I picked 10 keywords for my website and use them in right way to increase SEO.

• Quality of content – It is important to create high quality contents and also it should include the keywords and phrases.

• Blog for backlink – I made Blog page about findability things and put that in my site to get back link.

7. 404 page

• 404 page – It let my user know that there is an error with their request. It will recognize the mistake and guide them back to relevant pages on the site.
